2.2 How to upload a data set?

2.2.1 Grades dataset

Download: moody2022.csv

Grades in Professor Moody‚Äôs class data set.

Our working data set will be the Moody data set which stores data about students‚Äô grades in a large signature class taught by Professor Moody. The data set stores individual scores of students in class, their major, seniority and GPA. Data scientists may ask many questions such as, given the student‚Äôs score in class, does the final grade depend on the major and/or student‚Äôs seniority? For example, is it more difficult for computer science majors to earn an A, pass the class, than, say for students majoring in psychology? Does GPA play any role in grading? It should not - but maybe it does? We are still far away from being able to ask such questions, for now we will use Moody data set in code snippets which illustrates the core R functions which we will use in the active textbook.
